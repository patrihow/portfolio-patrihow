30 Jours JS

Défi personnel en JavaScript

J’ai conçu ce défi pour progresser avec régularité, apprendre en pratiquant, et documenter chaque étape de mon parcours.

Technologies: JavaScript (ES6+), HTML, CSS

Voir le dépôt GitHub

Détails du projet

Objectif

Structurer un apprentissage sur 30 jours avec des mini-projets quotidiens, en suivant des ressources variées (YouTube, Udemy, articles, docs officielles). Le défi est divisé en 3 niveaux progressifs :

  • Niveau 1 — Bases JavaScript : variables, fonctions, boucles, conditions, DOM, événements.
  • Niveau 2 — Concepts intermédiaires : manipulation d’API, promesses, localStorage, gestion d’erreurs.
  • Niveau 3 — Avancé : POO, design patterns, APIs Web (Canvas, WebSockets), async/await, animation, structure front-end.
